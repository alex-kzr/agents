## Правила применения

1. **Оцени тип задачи** перед тем как выбирать инструменты или подход.
2. **Если не уверен** — используй подход по умолчанию (строка 1): ответь через LLM дефолтного провайдера.
3. **Новые типы задач** добавляются в эту таблицу и обсуждаются с пользователем.
4. Таблица **расширяется** по мере накопления опыта — не добавляй типы задач самостоятельно, только после согласования.
