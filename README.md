# Agents Library

A curated catalog for autonomous AI agents.

## Structure

Skills are organized under `skills/` by domain.

Each skills catalog folder contains a `DESCRIPTION.md` listing its skills and sub-catalogs. 

Each skill is a self-contained directory with a `SKILL.md` defining when and how to invoke it, plus scripts, references, and templates.

The top-level index of all catalogs is at [`skills/DESCRIPTION.md`](skills/DESCRIPTION.md).
